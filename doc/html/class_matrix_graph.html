edge_tuple一条边的三元素元组
Returns

为了便于计算，添加边时并不是添加Edge类型，而是std::tuple<VIDType,VIDType,EWeightType>类型的值。

要求边的顶点均在[0,V)这个半闭半开区间。如果任何一个值超过该区间则认为顶点id无效，直接返回而不作添加
