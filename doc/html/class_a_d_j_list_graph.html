Detailed Description

template<int V>
class ADJListGraph< V >

图的邻接表表示，算法导论22章22.1节

图的邻接表主要包含一个数据：

  • _array：邻接表，类型为std::array<std::vector<std::pair<VIDType,EWeightType>>,V>，为V行，每一行代表一个节点
